Sobre o CodeTalks

O CodeTalks é uma plataforma fictícia de chat voltada para desenvolvedores, criada com foco em interface moderna, organização visual e interação entre usuários.

> informações acadêmicas_

Projeto desenvolvido para a disciplina de PPI (Programação para Internet) durante o 1° Bimestre .

> objetivo do projeto_

Interface moderna utilizando apenas front-end

O sistema foi desenvolvido utilizando exclusivamente tecnologias front-end, sem dependência de servidores ou compilação externa, seguindo os requisitos definidos para a disciplina.
